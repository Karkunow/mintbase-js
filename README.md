# Mintbase API
### WIP: Not for use!
General purpose Mintbase API for interacting with NEAR, Ethereum, Arweave and other supported blockchains and decentralized filestorage systems.

## API

- Fetching
    - Fetch marketplace
    - Fetch store(s)
    - Fetch user(s)
    - Fetch asset(s)
    - Fetch order(s)
- Wallet management
    - Login
    - Logout
    - Add accounts
    - Remove accounts
    - Switch accounts
- Mint
    - Mint token
    - Royalties
- Making offers
    - Bidding multiple assets
- Making listing / sell items
    - Split revenues
- Buying items
- Accepting offers
- Transfer items
- Advanced
    - Listening to events
    - Bulk transfer
    - Purchasing on others behalf
    - Using other fungible tokens (wNEAR, ...) instead of NEAR
    - Private (?) auctions 
    - Creating bundles
    - Schedule future listings
